Koneksi Google Sheets
Paste URL deployment dari Google Apps Script kamu di sini.
Cara Setup (3 Langkah)
1
Buka script.google.com → New Project. Hapus kode yang ada, paste seluruh isi file mishoa_appscript.gs.
2
Klik Deploy → New Deployment. Pilih type: Web App. Set Execute as: Me, dan Who has access: Anyone. Klik Deploy → salin URL-nya.
3
Paste URL di kolom di atas, klik Simpan & Tes Koneksi. Lalu update URL webhook di index.html toko kamu dengan URL yang sama.
